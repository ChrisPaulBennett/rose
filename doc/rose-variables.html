Rose Reference Guide: Variables

NPROC
Specifies the number of processors to run on. Default is 1.
ROSE_DATA
The path to the data directory of the running suite.
ROSE_DATAC
The path to the data directory of this cycle time in the running suite.
ROSE_DATAC????
The path to the data directory of the cycle time with an offset relative to the current cycle time. ???? is a duration:
  • A __ (double underscore) prefix denotes a cycle time in the future. Otherwise, it is a cycle time in the past.
  • nW denotes n weeks.
  • n or nD denotes n days.
  • Tn or TnH denotes n hours.
  • TnM denotes n minutes.
  • TnS denotes n seconds.
E.g. ROSE_DATACT6H is the data directory of 6 hours before the current cycle time.
E.g. ROSE_DATAC1D and ROSE_DATACT24H are both the data directory of 1 day before the current cycle time.
ROSE_LAUNCH_INNER
Specifies an inner wrapper to be run by launcher, can be set to "" to call the executable directly. Default is $0 --inner.
ROSE_LAUNCHER
Specifies the launcher program to run the prog.
ROSE_LAUNCHER_LIST
Specifies an alternative list of launchers.
ROSE_LAUNCHER_MPICH
Specifies the mpich launcher program to run the prog. Default is mpiexec.
ROSE_LAUNCHER_PREOPTS
Specifies the options to the launcher program to be inserted before the program name.
ROSE_LAUNCHER_POSTOPTS
Specifies the options to the launcher program to be inserted after the program name.
ROSE_LAUNCHER_ULIMIT_OPTS
Specifies the arguments to ulimit. Setting this variable to -a -s unlimited -d unlimited -a results in ulimit -a; ulimit -s unlimited; ulimit -d unlimited; ulimit -a
ROSE_ORIG_HOST
The name of the host where the rose suite-run command was invoked.
ROSE_SUITE_DIR
The path to the root directory of the running suite.
ROSE_SUITE_NAME
The name of the running suite.
ROSE_TASK_APP
Specify a named application configuration.
ROSE_TASK_CYCLE_TIME
The cycle time of the suite task, if there is one.
ROSE_TASK_LOG_ROOT
The root path for log files of the suite task.
ROSE_TASK_N_JOBS
The number of jobs to run in parallel in fcm make. (default=4)
ROSE_TASK_NAME
The name of the suite task.
ROSE_TASK_OPTIONS
Additional options and arguments for fcm make or rose app-run. (default='')
ROSE_TASK_PRE_SCRIPT
Source (. invoke) the specified script before running the main command. (default='')
ROSE_TASK_PREFIX
The prefix in the task name.
ROSE_TASK_SUFFIX
The suffix in the task name.
ROSE_TASK_UTIL
Specify a named task utility.
