Rose User Guide: Quiz

Introduction

This is a test to see how much Rose/cylc knowledge you've absorbed.

You should be familiar with all the content in the Rose User Guide.

Questions

  1. Rose is:
    a framework for growing roses outside scientific institutions
    a framework for getting legal advice
    a framework for managing and running suites of scientific applications
    a framework for developing web applications

    Answer: A framework for managing and running suites of scientific applications

  2. cylc is:
    a washing machine program
    a workload scheduler
    a framework for configuring Rose applications
    a suite engine

    Answer: a suite engine

  3. A user configuration for Rose is stored in:
    ~/.metomi/rose.conf
    ~/.rose/rose-user.conf
    ~/.rose.rc
    ~/.met-um/rose.rc

    Answer: ~/.metomi/rose.conf

  4. You can create a new suite with:
    rose suite-create
    rosie create
    fcm create
    rosie suite-create

    Answer: rosie create

  5. The sub-directory of a suite that holds Rose application configurations is called:
    rose-apps/
    app/
    apps/
    etc/

    Answer: app/

  6. rose-suite.conf files are for:
    reading in by cylc
    storing application input data
    identifying the suite with metadata
    jinja2 templating and installing files

    Answer: jinja2 templating and installing files

  7. rose edit cannot:
    commit changes to version control
    generate panels without metadata
    automatically fix trigger state errors
    load a single application by itself

    Answer: commit changes to version control

  8. Which metadata option best matches the following description?

    Used to order and group pages and variables in the config editor
    duplicate
    range
    sort-key
    title

    Answer: sort-key

  9. Which metadata option best matches the following description?

    Specify a regular expression that the value of the setting must match.




    Answer: pattern

  10. Which metadata option best matches the following description?

    If not present, the setting is assumed to be a scalar. A positive integer defines a fixed length array.
A colon : defines a dynamic length array.




    Answer: length

  11. I have the following metadata for one of my inputs:

    [namelist:foo=bar]
fail-if = this > 0 and (all(namelist:foo=baz == 0))

    Which application configuration settings would cause the fail-if checking to throw an error?





    Answer: bar = 1, baz = 0, 0 - it throws an error if the expression evaluates true - i.e. bar > 0 AND all elements in baz are 0. Missing options, as in B, cause the check to be skipped.

  12. I have a flawed metadata file, that looks like this:

    [namelist:foo=bar]
description=Bar, for controlling foo things.
help=Bar, part of the legendary made up variable family of foo, bar,
    =baz, qux, quux, corge, grault, garply, waldo, wibble, wobble,
    =wubble, fred, and flob.
    =
    =See http://en.wikipedia.org/wiki/Metasyntactic_variable .
trigger=namelist:spam=eggs: this < 0
type=integer
values=-1, 2, 10
warn-if=True

    Which metadata option is redundant?





    Answer: type - the values option supersedes other value definition options such as type, range, and pattern

  13. Which metadata option will cause an error?





    Answer: trigger - the option namelist:spam=eggs is not defined in the metadata

  14. Rose macros are used to:




    Answer: Check and apply changes to Rose configurations

  15. Output for Rose suites is usually under:
  16. The share/ subdirectory in a suite's output directory is for:




    Answer: any files that need to be shared between tasks

  17. Running the rosie go command:




    Answer: launches a discovery GUI

  18. We're searching for suites using rosie. We want every suite with Case Study in the title and where either the owner or the commit author was bob, across revisions. What is the correct command to run?





    Answer: rosie lookup --allrevs -Q ...

  19. Task A runs if B succeeds and C fails. Which is the correct suite.rc graph configuration entry?





    Answer: B & C:fail => A

  20. Which of the following is a valid cylc suicide trigger?




    Answer: A => !B

  21. Which of the following Rose commands invokes an in-built analysis engine?




    Answer: rose ana
