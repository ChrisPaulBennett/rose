Rose Documentation

Rose User Guide: Quiz

Introduction

This is a test to see how much Rose/cylc knowledge you've absorbed.

You should be familiar with all the content in the Rose User Guide.

Questions

  1. Rose is:
    • A) A framework for growing roses outside scientific institutions
    • B) A framework for getting legal advice
    • C) A framework for managing and running suites of scientific applications
    • D) A framework for developing web applications

    Answer: C) A framework for managing and running suites of scientific applications

  2. cylc is:
    • A) a washing machine program
    • B) a workload scheduler
    • C) a framework for configuring Rose applications
    • D) a suite engine

    Answer: D) a suite engine

  3. A user configuration for Rose is stored in:
    • A) ~/.metomi/rose.conf
    • B) ~/.rose/rose-user.conf
    • C) ~/.rose.rc
    • D) ~/.met-um/rose.rc

    Answer: A) ~/.metomi/rose.conf

  4. You can create a new suite with:
    • A) rose suite-create
    • B) rosie create
    • C) fcm create
    • D) rosie suite-create

    Answer: B) rosie create

  5. The sub-directory of a suite that holds Rose application configurations is called:
    • A) rose-apps/
    • B) app/
    • C) apps/
    • D) etc/

    Answer: B) app/

  6. rose-suite.conf files are for:
    • A) reading in by cylc
    • B) storing application input data
    • C) identifying the suite with metadata
    • D) jinja2 templating and installing files

    Answer: D) jinja2 templating and installing files

  7. rose edit cannot:
    • A) commit changes to version control
    • B) generate panels without metadata
    • C) automatically fix trigger state errors
    • D) load a single application by itself

    Answer: A) commit changes to version control

  8. Which metadata option best matches the following description?

    Used to order and group pages and variables in the config editor
    • A) duplicate
    • B) range
    • C) sort-key
    • D) title

    Answer: C) sort-key

  9. Which metadata option best matches the following description?

    Specify a regular expression that the value of the setting must match.
    • A) values
    • B) fail-if
    • C) range
    • D) pattern

    Answer: D) pattern

  10. Which metadata option best matches the following description?

    If not present, the setting is assumed to be a scalar. A positive integer defines a fixed length array.
A colon : defines a dynamic length array.
    • A) type
    • B) length
    • C) range
    • D) values

    Answer: B) length

  11. I have the following metadata for one of my inputs:

    [namelist:foo=bar]
fail-if = this > 0 and (all(namelist:foo=baz == 0))

    Which application configuration settings would cause the fail-if checking to throw an error?

    • A) 
      [namelist:foo]
bar = 1
baz = 0, 0
    • B) 
      [namelist:foo]
bar = 1
    • C) 
      [namelist:foo]
bar = 0
baz = 1, 0
    • D) 
      [namelist:foo]
bar = 0
baz = 1, 1

    Answer: A) - it throws an error if the expression evaluates true - i.e. bar > 0 AND all elements in baz are 0. Missing options, as in B, cause the check to be skipped.

  12. I have a flawed metadata file, that looks like this:

    [namelist:foo=bar]
description=Bar, for controlling foo things.
help=Bar, part of the legendary made up variable family of foo, bar,
    =baz, qux, quux, corge, grault, garply, waldo, wibble, wobble,
    =wubble, fred, and flob.
    =
    =See http://en.wikipedia.org/wiki/Metasyntactic_variable .
trigger=namelist:spam=eggs: this < 0
type=integer
values=-1, 2, 10
warn-if=True

    Which metadata option is redundant?

    • A) description
    • B) help
    • C) type
    • D) warn-if

    Answer: C) type - the values option supersedes other value definition options such as type, range, and pattern

  13. Which metadata option will cause an error?

    • A) description
    • B) help
    • C) trigger
    • D) type

    Answer: C) trigger - the option namelist:spam=eggs is not defined in the metadata

  14. Rose macros are used to:
    • A) Run applications
    • B) Record and play actions in the config editor
    • C) Check and apply changes to Rose configurations
    • D) Remove the need for metadata configuration

    Answer: C) Check and apply changes to Rose configurations

  15. Output for Rose suites is usually under:
    • A) $HOME/cylc-run/
    • B) $TMPDIR/suite-run/
    • C) $DATADIR/suite-run/
    • D) $HOME/rose-suite-run/

    Answer: A) $HOME/cylc-run/

  16. The share/ subdirectory in a suite's output directory is for:
    • A) any files that need to be shared between tasks
    • B) task working directories, e.g. containing Rose application namelists and other temporary files
    • C) task scripts and their output and error
    • D) the rose-suite.version file, which records any version control information about the suite, including any local modifications

    Answer: A) any files that need to be shared between tasks

  17. Running the rosie go command:
    • A) starts a suite running using cylc run
    • B) launches the output viewer
    • C) sshes to a cylc server
    • D) launches a discovery GUI

    Answer: D) launches a discovery GUI

  18. We're searching for suites using rosie. We want every suite with Case Study in the title and where either the owner or the commit author was bob, across revisions. What is the correct command to run?

    • A) rosie ls --title='Case Study' --username=bob --all
    • B) rosie lookup --allrevs 'Case Study' bob
    • C) rosie ls --all -Q 'Case Study' in title and 'bob' from \[author, owner\]
    • D) rosie lookup --allrevs -Q title contains 'Case Study' and \( owner eq bob or author eq bob \)

    Answer: D)

  19. Task A runs if B succeeds and C fails. Which is the correct suite.rc graph configuration entry?

    • A) 
              graph = """
            A => B & C:fail
        """
    • B) 
              graph = """
            B => C => A
        """
    • C) 
              graph = """
            B & C:fail => A
        """
    • D) 
              graph = """
            B => A
            C:fail => A
        """

    Answer: C) B & C:fail => A

  20. Which of the following is a valid cylc suicide trigger?
    • A) 
            graph = """
          A:kill => B
      """
    • B) 
            graph = """
          A murders B
      """
    • C) 
            graph = """
          A => !B
      """
    • D) 
            graph = """
          A:fail => B
      """

    Answer: C) A => !B

  21. Which of the following Rose commands invokes an in-built analysis engine?
    • A) rose mpi-launch
    • B) rose macro
    • C) rose ana
    • D) rose suite-hook

    Answer: C) rose ana
